安装环境
-----------

安装 Haskell 编译器:

.. code:: bash

    # OS X
    brew install ghc
    # Linux (Debian)
    sudo apt-get install ghc ghci

安装 Hackage 包管理器 opam:

.. code:: bash

    # OS X
    brew install opam
    opam update
    opam install core network


代码编译
-----------

编译二进制或者字节码:

.. code:: bash

    ghc -o helloworld helloworld.hs

进入解释器交互模式:

.. code:: bash

    ghci
